**observability.service.ts** â€” `@Injectable()` class `ObservabilityService`. Inject `DataSource` via `@InjectDataSource()`. All methods use raw SQL (pattern from research document). No TypeORM QueryBuilder.

Methods:

`async getLatencyByStep()`:
```sql
SELECT "skillKey" AS step_key,
       ROUND(AVG("durationMs")) AS avg_latency_ms,
       COUNT(*) AS total_executions,
       SUM(CASE WHEN status = 'failed' THEN 1 ELSE 0 END) AS error_count
FROM step_execution
WHERE "createdAt" > NOW() - INTERVAL '30 days'
GROUP BY "skillKey"
ORDER BY avg_latency_ms DESC
```

`async getCostByTicket()`:
```sql
SELECT te."complaintId",
       ROUND(SUM(tu."estimatedCostUsd")::numeric, 6) AS total_cost_usd,
       SUM(tu."totalTokens") AS total_tokens
FROM ticket_execution te
INNER JOIN step_execution se ON se."ticketExecutionId" = te.id
INNER JOIN token_usage tu ON tu."stepExecutionId" = se.id
GROUP BY te."complaintId"
ORDER BY total_cost_usd DESC
LIMIT 50
```

`async getErrorRateBySkill()`:
```sql
SELECT "skillKey",
       COUNT(*) AS total,
       SUM(CASE WHEN status = 'failed' THEN 1 ELSE 0 END) AS errors,
       ROUND(100.0 * SUM(CASE WHEN status = 'failed' THEN 1 ELSE 0 END) / NULLIF(COUNT(*),0), 2) AS error_rate_pct
FROM step_execution
GROUP BY "skillKey"
ORDER BY error_rate_pct DESC
```

`async getHitlRate()`:
```sql
SELECT
  COUNT(*) FILTER (WHERE status = 'waiting_human') AS hitl_count,
  COUNT(*) AS total_count,
  ROUND(100.0 * COUNT(*) FILTER (WHERE status = 'waiting_human') / NULLIF(COUNT(*),0), 2) AS hitl_rate_pct
FROM step_execution
```
Return single row as object.

`async getConformanceByTipologia()`:
```sql
SELECT t.key AS tipology_key,
       t.name AS tipology_name,
       ROUND(AVG((a.content->>'complianceScore')::float)::numeric, 2) AS avg_compliance_score,
       COUNT(a.id) AS evaluated_count
FROM artifact a
INNER JOIN step_execution se ON a."stepExecutionId" = se.id
INNER JOIN ticket_execution te ON se."ticketExecutionId" = te.id
INNER JOIN complaint c ON te."complaintId" = c.id
INNER JOIN tipology t ON c."tipologyId" = t.id
WHERE a."artifactType" = 'compliance_evaluation'
  AND (a.content->>'complianceScore') IS NOT NULL
GROUP BY t.key, t.name
ORDER BY avg_compliance_score DESC
```

`async getTokenTotals()`:
```sql
SELECT
  SUM("totalTokens") AS total_tokens,
  SUM("promptTokens") AS prompt_tokens,
  SUM("completionTokens") AS completion_tokens,
  ROUND(SUM("estimatedCostUsd")::numeric, 4) AS total_cost_usd,
  COUNT(*) AS total_calls
FROM token_usage
```
Return single row.

`async getExecutionTrace(execId: string)`:
```sql
SELECT
  se.id, se."skillKey", se.status,
  se."startedAt", se."completedAt", se."durationMs",
  se."errorMessage", se."retryCount",
  COALESCE(json_agg(DISTINCT jsonb_build_object(
    'id', lc.id, 'modelId', lc."modelId", 'provider', lc.provider,
    'latencyMs', lc."latencyMs", 'promptVersion', lc."promptVersion"
  )) FILTER (WHERE lc.id IS NOT NULL), '[]') AS llm_calls,
  COALESCE(json_agg(DISTINCT jsonb_build_object(
    'id', a.id, 'artifactType', a."artifactType", 'createdAt', a."createdAt"
  )) FILTER (WHERE a.id IS NOT NULL), '[]') AS artifacts
FROM step_execution se
LEFT JOIN llm_call lc ON lc."stepExecutionId" = se.id
LEFT JOIN artifact a ON a."stepExecutionId" = se.id
WHERE se."ticketExecutionId" = $1
GROUP BY se.id
ORDER BY se."createdAt" ASC
```

`async getTicketLogs(complaintId: string)`:
```sql
SELECT al.id, al.action, al."userId", al.metadata, al."createdAt"
FROM audit_log al
INNER JOIN ticket_execution te ON al."ticketExecutionId" = te.id
WHERE te."complaintId" = $1
ORDER BY al."createdAt" ASC
```
